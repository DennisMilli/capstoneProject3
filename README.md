### Description
A dynamic blog web application that generates pages server-side. This project marks the transition from static frontend layouts to backend server logic and templating.

### Tech Stack

Node.js

Express.js

EJS (Embedded JavaScript templates)

Vanilla CSS

### Features

Dynamic server-side routing

Server-Side Rendering (SSR) of blog content

Reusable view templates for headers and footers
